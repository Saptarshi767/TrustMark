Security Policy

About TrustMark

TrustMark is an educational blockchain analysis platform that helps users understand Ethereum transaction patterns and address classifications. We are committed to maintaining the highest security standards and transparency.

What We Do

  • Public Data Analysis: We analyze publicly available blockchain data from the Ethereum network
  • Educational Purpose: Our platform is designed for learning and demonstration purposes
  • Address Classification: We provide insights into transaction patterns and address behaviors
  • Community Flagging: Users can flag suspicious transactions for community awareness

What We DON'T Do

  • No Private Key Handling: We never request, store, or process private keys
  • No Financial Transactions: We do not facilitate any financial transactions
  • No Personal Data Collection: We don't collect personal information beyond wallet addresses
  • No Malicious Activity: We do not engage in phishing, scamming, or any harmful activities

Data Sources

All blockchain data is sourced from:

  • Etherscan API (official Ethereum blockchain explorer)
  • Public Ethereum blockchain data
  • User-contributed flags and classifications

Security Measures

  • HTTPS Encryption: All communications are encrypted
  • Content Security Policy: Strict CSP headers prevent XSS attacks
  • Input Validation: All user inputs are validated and sanitized
  • Rate Limiting: API endpoints are protected against abuse
  • Secure Headers: HSTS, X-Frame-Options, and other security headers
Educational Platform Notice

TrustMark is an educational and demonstration platform. It is designed to help users learn about blockchain analysis and is not intended for production financial use. Always verify information independently and never share private keys or sensitive information.
