Installation Guide

1
Download the Extension

Click the download button above to get the latest version of TrustMark Chrome Extension.

trustmark_chrome_extension.zip

2
Extract the ZIP File

Extract the downloaded ZIP file to a folder on your computer. Remember the location!

Tip: Create a dedicated folder like "TrustMark Extension" in your Documents.
3
Open Chrome Extensions

In Google Chrome, navigate to:

chrome://extensions/

Or go to Menu ‚Üí More Tools ‚Üí Extensions

4
Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

Developer mode: ON

5
Load Unpacked Extension

Click "Load unpacked" and select the folder where you extracted the extension files.

Important: Select the folder containing manifest.json, not the ZIP file itself.
6
Verify Installation

You should see the TrustMark extension appear in your extensions list with a purple shield icon.

TrustMark - Ethereum Reputation

How to Use

Click the Extension Icon

Click the TrustMark shield icon in your Chrome toolbar to open the purple-themed popup.

Scan Current Page

Click "Scan Current Page" to detect all Ethereum addresses on the webpage you're viewing.

View Address Reputation

Addresses will be highlighted with colored badges:

  • Red - Flagged/Hacker addresses
  • Yellow - Suspicious addresses
  • Purple - Normal addresses

Troubleshooting

  • Make sure you extracted the ZIP file completely
  • Verify you selected the correct folder (containing manifest.json)
  • Check that Developer mode is enabled
  • Try refreshing the extensions page

  • Make sure the TrustMark Flask server is running
  • Check that the page contains Ethereum addresses (0x...)
  • Try refreshing the page and scanning again
  • Open browser console (F12) to check for errors

  • Make sure you downloaded the latest version
  • Try removing and reinstalling the extension
  • Clear browser cache and reload
  • Check if you have any conflicting extensions

Need Help?

If you encounter any issues with the TrustMark Chrome Extension, please:

  • Check the troubleshooting section above
  • Ensure your TrustMark server is running
  • Verify you have the latest version of Chrome
  • Test with the provided test page: test_extension_purple.html
